# Links and embeds

## Obsidian wiki-links

Note: Flowershow is configured to work with Obsidian shortest-path-possible wiki-links, but will also work with Obsidian absolute paths.

**Wiki-link with "shortest-possible" path:**

```
[[post-1]]
```

<div data-testid="obsidian-wiki-link-shortest">
[[post-1]]
</div>

**Wiki-link with absolute path:**

```
[[blog/post-1]]
```

<div data-testid="obsidian-wiki-link-absolute">
[[blog/post-1]]
</div>

**Wiki-link to root README:**

```
[[README]]
```

<div data-testid="obsidian-wiki-link-root-readme">
[[README]]
</div>

**Wiki-link to README in blog folder:**

```
[[blog/README]]
```

<div data-testid="obsidian-wiki-link-blog-readme">
[[blog/README]]
</div>

**Wiki-link with special characters:**

```
[[Post With.Special.Chars %&(1)]]
```

<div data-testid="obsidian-wiki-link-special-chars">
[[Post With.Special.Chars %&(1)]]
</div>

**Wiki-link to heading:**

```
[[#Links in JSX blocks]]
```

<div data-testid="obsidian-wiki-link-heading">
[[#Links in JSX blocks]]
</div>

**Wiki-link to asset:**

```
[[assets/sample.pdf]]
```

<div data-testid="obsidian-wiki-link-asset">
[[assets/sample.pdf]]
</div>

**Wiki-link case insensitive:**

```
[[POST-1]]
```

<div data-testid="obsidian-wiki-link-case">
[[POST-1]]
</div>

**Wiki-link to file with permalink:**

```
[[post-with-permalink]]
```

<div data-testid="obsidian-wiki-link-file-with-permalink">
[[post-with-permalink]]
</div>

**Wiki-link with alias:**

```
[[post-1|Some Alias]]
```

<div data-testid="obsidian-wiki-link-alias">
[[post-1|Some Alias]]
</div>

## CommonMark links

**Absolute path link:**

```
[/blog/post-1](/blog/post-1.md)
```

<div data-testid="common-mark-link-absolute">
[/blog/post-1](/blog/post-1.md)
</div>

**Relative path link:**

```
[syntax](syntax.md)
```

<div data-testid="common-mark-link-relative">
[syntax](syntax.md)
</div>

**Relative path link with dot-slash:**

```
[./syntax](./syntax.md)
```

<div data-testid="common-mark-link-relative-dot-slash">
[./syntax](./syntax.md)
</div>

**Link to root README with absolute path:**

```
[/README](/README.md)
```

<div data-testid="common-mark-link-root-readme-absolute">
[/README](/README.md)
</div>

**Link to parent directory README:**

```
[../README](../README.md)
```

<div data-testid="common-mark-link-parent-readme">
[../README](../README.md)
</div>

**External link:**

```
[External link](https://example.com)
```

<div data-testid="common-mark-link-external">
[External link](https://example.com)
</div>

**Link to heading:**

```
[#Links%20in%20JSX%20blocks](#Links%20in%20JSX%20blocks)
```

<div data-testid="common-mark-link-heading">
[#Links%20in%20JSX%20blocks](#Links%20in%20JSX%20blocks)
</div>

**Link to asset:**

```

```

<div data-testid="common-mark-link-asset">

</div>

**Link with spaces in path:**

```

```

<div data-testid="common-mark-link-with-spaces">

</div>

**Link with special characters:**

